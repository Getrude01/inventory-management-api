# Inventory Management API

## Project Overview
This project is a backend Inventory Management API built using Django and Django REST Framework.  
It allows authenticated users to manage inventory items by creating, viewing, updating, and deleting products while tracking stock levels.

The API simulates a real-world inventory system for a store.

---

## Features
- User authentication (JWT-based)
- Inventory item CRUD operations
- Category management
- Users can only manage their own inventory
- View current inventory levels
- Secure API endpoints

---

## Tech Stack
- Python
- Django
- Django REST Framework (DRF)
- SQLite (Development database)
- JWT Authentication
